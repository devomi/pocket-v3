WEBPANEL Setup

1. Web Server Side :

  1. Open WEBPANEL-v1.9->upload folder and Put the all files in upload folder on your host server (with php & mysql support)
  2. Create a database and Note the Database Name, Database User, Host and Database Password
  3. Open WEBPANEL-v1.9->upload Folder and Edit config/config.php File and change the MYSQL Connection with your access.
  4. Change all the data on config/config.php where Required
  5. Change all the data on fcm/connect.inc.php where Required
  6. After changing the values, you can Navigate to http://your-domain.com/install.php and create an Admin Account... all data tables are Created automatically.
  7. Note : if you are updating ,
    1. copy all the files to webserver(overwrite if asks)
    2. if you are updating webapenl from v1.9 to v2.0 then upload and overwrite all files in update folder to your server (overwrite). that's it.. nothing else
    3. if you are updating webapenl from v1.8 to v1.9 then upload and overwrite all files in update folder to your server (overwrite). that's it.. nothing else
    4. if you are updating webapenl from v1.7 to v1.8 then upload and overwrite all files in update folder to your server (overwrite). that's it.. nothing else
    5. if you are updating webapenl from v1.6 to v1.7 then upload All files to your server (overwrite). that's it.. nothing else
    6. if you are updating webapenl from v1.5 to v1.6 then upload All files to your server (overwrite)
    7. if you are updating webapenl from v1.4 to v1.5 then go to - yourdomain.com/update4-5.php
    8. if you are updating webapenl from v1.3 to v1.5 then go to - yourdomain.com/update3-5.php
    9. if you are updating webapenl from v1.3 to v1.4 then go to - yourdomain.com/update3-4.php
Note: You should use first/second level domain..like this..
http://example.com or http://folder.example.com
you should not use like this,,
http://example.com/folder

2. Android Side :

Edit Configuration File Config.java
  • Set your your URL on the Base_Url value. Here,our url is http://pocket.oxywebs.in/ so add your url in Base_Url and don't forget the slash / at the end of the url
